PropaNice PCs Website

Files included:
- index.html
- style.css
- script.js

How to host free:
1. Go to Netlify Drop, Cloudflare Pages, or GitHub Pages.
2. Upload this folder or connect it to a GitHub repository.
3. Set index.html as the homepage.

Editing:
- Change prices/builds in index.html.
- Change colours and layout in style.css.
- Change contact form behaviour in script.js.

This is a static website, so the form currently creates a message for the customer to copy.
